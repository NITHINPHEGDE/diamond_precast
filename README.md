# Diamond Precast Website

Premium precast construction company website built with React 19 + Vite + Tailwind CSS.

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- Swiper.js
- React Photo View
- React CountUp
- React Scroll
- React Helmet Async

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Image Replacement Guide

All placeholder images use picsum.photos. Search for `// REPLACE IMAGE` comments throughout the codebase to find every image location.

## Structure

```
src/
  components/       # All section components
  data/             # Static data arrays
  styles/           # Global CSS
  assets/           # Static assets
```
